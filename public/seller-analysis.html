Agent-to-Agent Transaction Patterns

The 5 dominant patterns driving 90%+ of agent-to-agent transaction volume

Pattern 1: Research Agent → Data Sellers

Research agents buy web scraping, news, and market data on-demand. Highest frequency pattern.

Agent → Firecrawl ($0.01/scrape)
Agent → Gloria AI ($0.03/news)
Agent → Minifetch ($0.002/meta)

Pattern 2: Trading Agent → Intel Sellers

DeFi/trading agents buy whale tracking, smart money signals, and risk scores before executing trades.

Agent → SLAMai ($0.001/signal)
Agent → Einstein AI ($0.01/query)
Agent → Heurist ($0.005/analysis)

Pattern 3: Content Agent → LLM Gateways

Agents generating text, images, or audio buy inference from LLM gateways without managing API keys.

Agent → BlockRun ($0.003/call)
Agent → Spraay ($0.005/chat)
Agent → AIsa ($0.01/1K tokens)

Pattern 4: DeFi Agent → Security Sellers

Agents verify contract safety before interacting. Critical safety loop for autonomous fund management.

Agent → AgentLisa ($0.50/audit)
Agent → Cybercentry ($0.02/check)
Agent → x402-secure (risk score)

Pattern 5: Orchestrator → Specialist Agents

Coordinator agents decompose complex tasks and hire specialist agents for each subtask.

Questflow → weather ($0.001)
Questflow → data ($0.005)
Questflow → analysis ($0.01)

Pattern 6: Trading Agent → Execution

Agents that need to execute on-chain operations buy swap quotes, trade execution, and settlement.

Agent → Elsa swap ($0.02)
Agent → AEON settle (real-world)
Agent → Spraay batch ($0.01)

What Makes Seller Agents Successful

7 patterns shared by the highest-volume sellers in the x402 ecosystem
  • 1
    Sub-Cent Pricing Sweet Spot

    3,500+ services (the largest bracket) price between $0.001-$0.01 per call. This was impossible with credit cards ($0.30 fixed fee). x402 settlement on Base costs ~$0.0001, making penny transactions profitable. Top sellers like Minifetch ($0.002) and SLAMai ($0.001) thrive here.

  • 2
    Payment = Authentication (Zero Friction)

    The most successful sellers require ZERO setup: no accounts, no API keys, no OAuth flows, no KYC. The agent's USDC payment IS the authentication. AIsa, BlockRun, and Firecrawl all follow this pattern -- any wallet-equipped agent can start using the service immediately.

  • 3
    Machine-Readable Output

    Top sellers return structured, machine-consumable data -- not human-readable pages. Gloria AI returns categorized news signals for trading stacks. Heurist returns composable data objects. Elsa returns typed portfolio analytics. Agents can act on the output without parsing.

  • 4
    Discoverability via x402 Bazaar

    Successful sellers set discoverable: true in route config, include mimeType and outputSchema descriptions, and list on x402scan, x402list.fun, and x402.org ecosystem directory. Agents programmatically find and evaluate services before buying.

  • 5
    Aggregation Over Specialization

    The two highest-volume sellers (AIsa at 22M+ tx, x402engine at 60 endpoints) are aggregators. They bundle diverse capabilities behind one payment interface. Agents prefer one provider for 10 tasks over 10 providers for 1 task each -- fewer integrations, fewer wallets to manage.

  • 6
    Safety-Critical Services Command Premium Pricing

    AgentLisa charges $0.50-$5.00/scan (100x more than data queries) because the service is safety-critical: agents MUST verify contracts before depositing funds. Cybercentry, x402-secure, and BlackSwan occupy the same "must-have" category. Security = inelastic demand.

  • 7
    Near-Zero Artificial Activity

    Artemis research shows 47% of all x402 transactions are teams gaming leaderboards. The top legitimate sellers (AIsa, x402-secure, LucyOS, AgentLisa, Heurist) are distinguished by millions of micropayments with near-zero artificial activity -- real utility, not inflated metrics.
